# Galenite Bothive page

Открывать: `index.html`.

## Ассеты

Положи свои реальные ассеты в папку `assets` с такими именами:

- `laptop.gif` — гифка ноута
- `paperclip.gif` — гифка скрепки
- `stars.gif` — маленькая гифка звезд после создания
- `id.svg` — иконка поля имени бота
- `key.svg` — иконка поля ключа
- `settings.svg` и `bell.svg` — иконки в шапке, уже лежат как плейсхолдеры

## Что поправлено

- Плашка формы всегда открывается строго в центре экрана: по горизонтали и вертикали.
- Форма больше не режется контейнером из-за неправильного смещения.
- Гифка при открытии формы уходит выше, чтобы не залезать на карточку.
- Анимации переписаны на плавные `transform`, `opacity`, `width/height` с мягким easing.
- Добавлен каскадный вход элементов формы.
- После создания показывается `stars.gif`, потом карточка схлопывается обратно в компактную плашку бота.
- Добавлен адаптив под большие мониторы, MacBook, низкие экраны и телефоны.
